# Nastavení IDE

V tomto krátkém článku se dozvíte, jak jsem si upravil můj hlavní editor, kterým je aktuálně **Visual Studio Code**.

## Základ

Font jsem neměnil - používám tedy výchozí font Visual Studia Code kterým je **Consolas**. Velikost fontu mám nastavenou na **18** hlavně kvůli streamování na Twitch.

## Vzhled

Mám dva oblíbené "vzhledy" editoru kterými jsou [Ayu Mirage](https://marketplace.visualstudio.com/items?itemName=teabyii.ayu) a [Moonlight](https://marketplace.visualstudio.com/items?itemName=atomiks.moonlight). Ikony u souborů a složek, které představují to, v jakém jazyce jsou napsané nebo k čemu slouží, upravuje rozšíření [Material Icon Theme](https://marketplace.visualstudio.com/items?itemName=PKief.material-icon-theme).

## Rozšíření

Při vytváření nových **React komponentů a Next.js stránek** šetří hodně času rozšíření [ES7 React snippets](https://marketplace.visualstudio.com/items?itemName=dsznajder.es7-react-js-snippets) které podporuje různé typy syntaxe společně s možností použití **TypeScriptu**.

Kód mi formátuje [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) a kontroluje ho [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint).

Stylování stránek jsem samozřejmě začínal s klasickým CSS a zkoušel i Sass, dnes už ale používám v podstatě jen **TailwindCSS**. Přestože už spoustu tříd umím zpaměti tak se rozšíření [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) rozhodně hodí používat.

A zde jsou další podstatná rozšíření ke kterým už nemám co dodat:

- [Auto Rename Tag](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag)
- [GitLens — Git supercharged](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens)
- [PostCSS Language Support](https://marketplace.visualstudio.com/items?itemName=csstools.postcss)
- [Prisma](https://marketplace.visualstudio.com/items?itemName=Prisma.prisma)

Ještě mám spoustu rozšíření na intellisense a formátování individuálních jazyků a technologií, ty si ale můžete najít a nainstalovat podle vlastní potřeby.

## Závěr

Takhle tedy vypadá můj editor ve kterém jsem napsal spoustu webů a webových aplikací **včetně této stránky**. Doufám, že vám tento článek nějakým způsobem pomohl ;)
